💻 氣泡排序法 - C語言寶例

void bubbleSort(int arr[], int n) {
for (int i = 0; i < n - 1; i++) {
bool swapped = false;
for (int j = 0; j < n - i - 1; j++) {
if (arr[j] > arr[j + 1]) {
int temp = arr[j];
arr[j] = arr[j + 1];
arr[j + 1] = temp;
swapped = true;
}
}
if (!swapped) break;
}
}

📚 氣泡排序法原理

  • 基本概念： 氣泡排序是一種簡單的排序算法，通過重複走訪陣列，比較相鄰元素，如果順序不對則交換它們
  • 工作流程： 每一輪都會把最大（或最小）的元素"冒泡"到陣列的末尾（或開頭）
  • 輪數計算： 需要 n-1 輪才能完全排序，其中 n 是陣列元素數量
  • 穩定性： 氣泡排序是穩定的排序算法，相等元素的相對順序不會改變
  • 適用場景： 小規模資料集、教學用途、或資料基本已排序的情況
  • 演算法優化： 如果某一輪沒有發生交換，說明陣列已排序，可提前終止
