合併排序使用分治法：先遞歸分割陣列，再按順序合併已排序的子陣列。

💻 合併排序法 - C語言寶例

void merge(int arr[], int left, int mid, int right) {
int n1 = mid - left + 1;
int n2 = right - mid;
// 建立臨時陣列
int leftArr[n1], rightArr[n2];
for (int i = 0; i < n1; i++)
leftArr[i] = arr[left + i];
for (int j = 0; j < n2; j++)
rightArr[j] = arr[mid + 1 + j];
int i = 0, j = 0, k = left;
// 比較兩個子陣列
while (i < n1 && j < n2) {
if (leftArr[i] <= rightArr[j]) {
arr[k++] = leftArr[i++];
} else {
arr[k++] = rightArr[j++];
}
}
// 複製剩餘元素
while (i < n1)
arr[k++] = leftArr[i++];
while (j < n2)
arr[k++] = rightArr[j++];
}
void mergeSort(int arr[], int left, int right) {
if (left < right) {
int mid = left + (right - left) / 2;
// 分割左半部分並排序
mergeSort(arr, left, mid);
// 分割右半部分並排序
mergeSort(arr, mid + 1, right);
// 合併已排序的子陣列
merge(arr, left, mid, right);
}
}

📚 合併排序法原理

  • 分治策略： 合併排序將問題分解為更小的子問題，分別解決後再合併結果
  • 分割階段： 遞歸地將陣列分成越來越小的子陣列，直到每個子陣列只有一個元素
  • 合併階段： 從最小的子陣列開始，將已排序的子陣列逐步合併成更大的已排序陣列
  • 穩定性： 合併排序是穩定的，相等元素的相對順序不變
  • 複雜度優勢： 所有情況下時間複雜度都是 O(n log n)，性能穩定可靠
  • 應用場景： 大規模資料、外部排序、並行計算等。LinkedList 常使用合併排序
