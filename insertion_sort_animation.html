💻 插入排序法 - C語言寶例

void insertionSort(int arr[], int n) {
for (int i = 1; i < n; i++) {
int key = arr[i];
int j = i - 1;
while (j >= 0 && arr[j] > key) {
arr[j + 1] = arr[j];
j--;
}
arr[j + 1] = key;
}
}

📚 插入排序法原理

  • 基本概念： 插入排序將陣列分為已排序和未排序兩部分，每次從未排序部分取一個元素，插入到已排序部分的正確位置
  • 工作流程： 從第二個元素開始，與已排序部分從右至左比較，找到正確位置後插入（實際通常是後移而非插入）
  • 最好情況： 當陣列已排序時，只需 n-1 次比較，是 O(n) 複雜度
  • 穩定性： 插入排序是穩定的，相等元素的相對順序不變
  • 適用場景： 小規模陣列、幾乎已排序的資料、在線算法等
  • 實際應用： 撲克牌整理、數據庫索引等場景常用
