💻 選擇排序法 - C語言寶例

void selectionSort(int arr[], int n) {
for (int i = 0; i < n - 1; i++) {
int minIdx = i;
for (int j = i + 1; j < n; j++) {
if (arr[j] < arr[minIdx]) {
minIdx = j;
}
}
if (minIdx != i) {
int temp = arr[i];
arr[i] = arr[minIdx];
arr[minIdx] = temp;
}
}
}

📚 選擇排序法原理

  • 基本概念： 選擇排序通過在未排序部分中反覆選擇最小（或最大）元素，將其放到已排序部分的末尾
  • 工作流程： 每一輪都會在剩餘未排序的陣列中找到最小值，然後與起始位置交換
  • 時間特性： 無論初始順序如何，選擇排序的比較次數都是固定的：n×(n-1)/2
  • 穩定性： 選擇排序通常不穩定，因為交換可能改變相等元素的相對順序
  • 適用場景： 數據規模較小、記憶體有限、或交換操作成本低的情況
  • 優化特點： 交換次數少於氣泡排序，最多只需 n-1 次交換
