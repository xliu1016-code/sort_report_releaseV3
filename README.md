#學習心得  
在學習各種排序演算法的過程中，我逐漸從「寫程式」轉變成「理解背後邏輯」。 而且有時候不只是單單解決排序的問題，而是要考量到效率、複雜度、穩定度等等。從氣泡排序和選擇排序中可以了解到最直觀的交換與比較的概念，後續學到的插入與合併則帶出排序區間，漸漸讓我了解道從個別比較到建立有序的結構。在自己探索其他排序法的過程中，則讓我理解到分析它們的優缺點的重要性，以及在不同情境下做出選擇。排序演算法看似基礎，但其實訓練的是邏輯拆解、效率思維與問題建模能力，對未來學習更進階的資料結構與演算法有很大的幫助。
